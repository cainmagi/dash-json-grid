```python
index: int = sanitize_list_index(index: Any)
```

Try to ensure `index` to be a `int`. If failed, raise `ValueError`.

## Arguments

### Requires

| Argument | Type  |             Required              | <center>Description</center> |
| :------: | :---: | :-------------------------------: | ---------------------------- |
| `index`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | The value to be sanitized.   |

### Returns

| Argument | Type  | <center>Description</center> |
| :------: | :---: | :--------------------------- |
| `index`  | `int` | The sanitized `index`.       |
