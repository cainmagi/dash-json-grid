```python
data_selected: Any = get_item_of_object(data: Any, index: Any)
```

Run `data[index]` supposing that `data` is abitrary and `index` can be a one-value sequence.

:::warning

Note that `index` is not a [`route`](./Route.mdx). If it is a sequence, it should only
contain one element.

:::

## Arguments

### Requires

| Argument | Type  |             Required              | <center>Description</center>                                                                                               |
| :------: | :---: | :-------------------------------: | -------------------------------------------------------------------------------------------------------------------------- |
|  `data`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | The whole data to be indexed.                                                                                              |
| `index`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | The index to be used for finding a part of the data. It can be a number, or a keyword, or a one-value sequence like `[0]`. |

### Returns

|    Argument     | Type  | <center>Description</center>                         |
| :-------------: | :---: | :--------------------------------------------------- |
| `data_selected` | `Any` | The part of the `data` located by the given `index`. |
