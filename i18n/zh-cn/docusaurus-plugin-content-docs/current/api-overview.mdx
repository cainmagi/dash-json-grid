# API 总览

<APITopBar type="package" source="." />

本组件包是对React项目[React JSON Grid<IconExternalLink/>][git-react-json-grid]的Dash移植版。用以提供复杂JSON对象/数组（对应为Python字典/列表）的结构化、嵌套化网格视图。
