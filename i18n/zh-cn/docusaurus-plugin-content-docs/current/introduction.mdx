# 简介

Dash JSON Grid是一个[Dash组件<IconExternalLink/>][link-dash-component]包。

本组件包是对React项目[React JSON Grid<IconExternalLink/>][git-react-json-grid]的Dash移植版。
用以提供复杂JSON对象/数组（对应为Python字典/列表）的结构化、嵌套化网格视图。

以下两图对比了原React项目和本项目的范例。由于本项目仅仅是原React组件的Dash封装，它们的效果可以
视作是等价的。

|                                      React JSON Grid                                       |                                      Dash JSON Grid                                      |
| :----------------------------------------------------------------------------------------: | :--------------------------------------------------------------------------------------: |
| <img alt="demo-react" className="ondark-inv" src={rawURL("main/display/demo-react.png")}/> | <img alt="demo-dash" className="ondark-inv" src={rawURL("main/display/demo-dash.png")}/> |

## React JSON Grid的简单范例 {#a-simple-live-demo-of-react-json-grid}

:::warning

注意此处的范例纯粹是用React JSON Grid写就。Dash JSON Grid仅仅是React JSON Grid的封装，因此
可以认为Dash JSON Grid能实现和以下范例相同的效果。然而，由于以下范例是由文档的全局依赖项
<AppVer/>构建，其与旧版的Dash JSON Grid所依赖的版本可能不同，故而，该范例未必能确切地和你正在
检索的Dash JSON Grid版本对应上。

:::
