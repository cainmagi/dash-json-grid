`ThemeConfigs`是`TypedDict`.

用来定制主题的详细配置。

## 别名 {#aliases}

该类型可以按以下方式之一获取

```python
import dash_json_grid as djg

```

## 关键字 {#keywords}

|          关键字          | 类型  | 必选 | <center>说明</center>                |
| :----------------------: | :---: | :--: | :----------------------------------- |
|        `bgColor`         | `str` |      | 背景颜色                             |
|      `booleanColor`      | `str` |      | `bool`类型数据的文字颜色。           |
|    `cellBorderColor`     | `str` |      | 表格单元的背景颜色。                 |
|    `highlightBgColor`    | `str` |      | 选中某部分时，其高亮状态背景色。     |
|       `indexColor`       | `str` |      | 序列下标的文字颜色。                 |
|      `keyNameColor`      | `str` |      | JSON关键字的文字颜色。              |
|      `numberColor`       | `str` |      | 数值类型数据的文字颜色。             |
|      `objectColor`       | `str` |      | 无法识别对象的文字颜色。             |
| `searchHighlightBgColor` | `str` |      | 搜索关键字时，被搜索高亮的背景颜色。 |
|      `stringColor`       | `str` |      | 字符串类型数据的文字颜色。           |
|    `tableBorderColor`    | `str` |      | 表格边框的颜色。                     |
|   `tableHeaderBgColor`   | `str` |      | 表头部分的背景颜色。                 |
|    `tableHeaderColor`    | `str` |      | 表头部分的文字颜色。                 |
