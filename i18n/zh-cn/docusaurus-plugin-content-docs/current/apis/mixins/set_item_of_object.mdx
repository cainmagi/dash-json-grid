```python
set_item_of_object(data: Any, index: Any, value: Any)
```

对于任意数据`data`，赋值`data[index] = value`，其中`index`可以是一个由单元素构成的序列。

若`data`的成员无法被设置，抛出`ValueError`。

若需要修改的`data`是序列，但`value`无法成功广播到`data`的成员上，抛出`IndexError`。

:::warning

注意`index`并非是[路由（`route`）](./Route.mdx)类型。若该值是序列，该序列只能包含单元素。

:::

## 参数 {#arguments}

### 输入 {#requires}

|  参数   | 类型  |               必选                | <center>说明</center>                                                                     |
| :-----: | :---: | :-------------------------------: | ----------------------------------------------------------------------------------------- |
| `data`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 需要索引的完整数据。                                                                      |
| `index` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 用来索引并更新数据某一部分的下标值。该值可以是数字、关键字、或者是形如`[0]`的单元素序列。 |
| `value` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 用来更新`data`被`index`选中部分的新值。                                                   |

            {"id": "1003", "type": "Blueberry"},
            {"id": "1004", "type": "Devil's Food"},
        ]
    },

        {"id": "5005", "type": "Sugar"},
