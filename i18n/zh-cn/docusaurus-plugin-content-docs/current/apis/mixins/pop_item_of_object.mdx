```python
data_selected: Any = pop_item_of_object(data: Any, index: Any)
```

对于任意数据`data`，取得`val = data[index]; del data[index]; return val`的结果，
其中`index`可以是一个由单元素构成的序列。

:::warning

注意`index`并非是[路由（`route`）](./Route.mdx)类型。若该值是序列，该序列只能包含单元素。

:::

## 参数 {#arguments}

### 输入 {#requires}

|  参数   | 类型  |               必选                | <center>说明</center>                                                                     |
| :-----: | :---: | :-------------------------------: | ----------------------------------------------------------------------------------------- |
| `data`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 需要索引的完整数据。                                                                      |
| `index` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 用来索引并删除数据某一部分的下标值。该值可以是数字、关键字、或者是形如`[0]`的单元素序列。 |

### 输出 {#returns}

|      参数       | 类型  | <center>说明</center>                                 |
| :-------------: | :---: | :---------------------------------------------------- |
| `data_selected` | `Any` | 由`index`定位、并从`data`删除并弹出的`data`的一部分。 |

```python title="remove_a_part_of_data.py" showLineNumbers
import pprint
from dash_json_grid.mixins import pop_item_of_object

data = {
    "id": "0001",
    "type": "donut",
    "name": "Cake",
    "ppu": 1111.55,
    "batters": {
        "batter": [
            {"id": "1001", "type": "Regular"},
            {"id": "1002", "type": "Chocolate"},
            {"id": "1003", "type": "Blueberry"},
            {"id": "1004", "type": "Devil's Food"},
        ]
    },
    "topping": [
        {"id": "5001", "type": "None"},
        {"id": "5002", "type": "Glazed"},
        {"id": "5005", "type": "Sugar"},
        {"id": "5007", "type": "Powdered Sugar"},
        {"id": "5006", "type": "Chocolate with Sprinkles"},
        {"id": "5003", "type": "Chocolate"},
        {"id": "5004", "type": "Maple"},
    ],
}

route = "topping"

removed = pop_item_of_object(data, route)
pprint.pprint(data)
pprint.pprint(removed)
```
