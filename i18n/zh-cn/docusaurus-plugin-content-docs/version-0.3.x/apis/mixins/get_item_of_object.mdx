```python
data_selected: Any = get_item_of_object(data: Any, index: Any)
```

对于任意数据`data`，获取`data[index]`的值，其中`index`可以是一个由单元素构成的序列。

:::warning

注意`index`并非是[路由（`route`）](./Route.mdx)类型。若该值是序列，该序列只能包含单元素。

:::

## 参数 {#arguments}

### 输入 {#requires}

|  参数   | 类型  |               必选                | <center>说明</center>                                                               |
| :-----: | :---: | :-------------------------------: | ----------------------------------------------------------------------------------- |
| `data`  | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 需要索引的完整数据。                                                                |
| `index` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 用来索引数据某一部分的下标值。该值可以是数字、关键字、或者是形如`[0]`的单元素序列。 |

### 输出 {#returns}

|      参数       | 类型  | <center>说明</center>           |
| :-------------: | :---: | :------------------------------ |
| `data_selected` | `Any` | 由`index`定位的`data`的一部分。 |
