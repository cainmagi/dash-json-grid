```python
index: int = sanitize_list_index(index: Any)
```

尝试确保某下标`index`的类型为`int`。若失败，抛出`ValueError`。

## 参数 {#arguments}

### 输入 {#requires}

|  参数   | 类型  |               必选                | <center>说明</center> |
| :-----: | :---: | :-------------------------------: | --------------------- |
| `index` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 要处理的值。          |

### 输出 {#returns}

|  参数   | 类型  | <center>说明</center> |
| :-----: | :---: | :-------------------- |
| `index` | `int` | 处理后的`index`。     |
