```python
flag: TypeGuard[Sequence[Any]] = is_sequence(val: Any)
```

检查某值`val`是否是序列。

## 参数 {#arguments}

### 输入 {#requires}

| 参数  | 类型  |               必选                | <center>说明</center> |
| :---: | :---: | :-------------------------------: | --------------------- |
| `val` | `Any` | <InlineIcon icon={vsiCheckIcon}/> | 要验证的值。          |

### 输出 {#returns}

|  参数  |  类型  | <center>说明</center>                           |
| :----: | :----: | :---------------------------------------------- |
| `flag` | `bool` | 若输出`True`，则`val`的类型是序列，反之则不是。 |
