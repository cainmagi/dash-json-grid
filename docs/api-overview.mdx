# Overview of APIs

<APITopBar type="package" source="__init__.py" />

Dash porting version of the react project
[React JSON Grid<IconExternalLink/>][git-react-json-grid]. Provide structured and
nested grid table view of complicated JSON objects/arrays.

<ThemedComponent
  className="mb-default"
  light={<OverviewLightSVG />}
  dark={<OverviewDarkSVG />}

This graph shows the high-level overview of the whole package. The core component is
`DashJsonGrid` which has been already modified by merging mixins.
