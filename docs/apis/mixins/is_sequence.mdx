Check whether `val` is a sequence or not.

## Arguments

### Requires

| Argument | Type  |             Required              | <center>Description</center> |
| :------: | :---: | :-------------------------------: | ---------------------------- |
|  `val`   | `Any` | <InlineIcon icon={vsiCheckIcon}/> | The value to be verified.    |

### Returns

| Argument |  Type  | <center>Description</center>                              |
| :------: | :----: | :-------------------------------------------------------- |
|  `flag`  | `bool` | If this value is `True`, `val` is a sequence, vice versa. |
